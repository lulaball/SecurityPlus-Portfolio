# Lab 1 - Password Cracking with John the Ripper

## Objective
Learn password cracking with John the Ripper.

## Tools Used
- Kali Linux
- John the Ripper

## Steps
1. Install John the Ripper.
2. Create a simple password-protected file.
3. Use John to crack the password.

## Reflection
This lab showed me how easy weak passwords are to crack.
